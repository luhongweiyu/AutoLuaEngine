# 构建与运行

## 构建 APK

```powershell
$env:JAVA_HOME='D:\soft\Android\Android Studio\jbr'
$env:PATH="$env:JAVA_HOME\bin;$env:PATH"
.\gradlew.bat :app:assembleDebug
```

工作目录：

```text
engines/android
```

## 脚本目录

Android 脚本统一存放在：

```text
/sdcard/AutoLuaEngine/scripts/
```

首次启动会申请脚本存储访问：Android 11 及以上打开“所有文件访问”系统页，Android 10
及以下申请传统读写存储权限。授权后直接创建该目录，并复制缺失的内置示例文件，不覆盖
已有文件。脚本列表、悬浮窗运行入口和引擎读取的都是这个共享目录中的真实文件，外部编辑器
保存后返回 App 即可刷新。

```lua
local w, h, pixels = m.capture()
```

完整参数、返回值和兼容命名空间说明见 [脚本文档](脚本文档.md)。新增脚本 API 必须先
实现 `libengine.so/core/api` 真实逻辑，再通过 `system_c_api` 暴露 C ABI，最后接
Lua/JS/Go 语言绑定。
