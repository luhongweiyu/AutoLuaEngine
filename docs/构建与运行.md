# 构建与运行

## 构建 APK

```powershell
$env:JAVA_HOME='D:\soft\Android\Android Studio\jbr'
$env:PATH="$env:JAVA_HOME\bin;$env:PATH"
.\gradlew.bat :app:assembleDebug
```

工作目录：

```text
engines/android
```

## 当前运行范围

App/UI、EngineService、HTTP 通讯、脚本运行、停止、日志保留。

当前脚本层只保留：

```lua
print(...)
m.sleep(ms)
m.log.print(text)
m.capture()
m.keepCapture()
m.releaseCapture()
m.setCaptureCacheMs(ms)
```

## 已清空旧能力

旧的点击、滑动、按键等自动化入口已清空。新增脚本 API 必须先实现
`libengine.so/core/api` 真实逻辑，再通过 `system_c_api` 暴露 C ABI，最后接 Lua/JS/Go
语言绑定。
