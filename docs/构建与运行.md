# 构建与运行

## 构建 APK

```powershell
$env:JAVA_HOME='D:\Android\openjdk\jdk-17.0.8.101-hotspot'
$env:PATH="$env:JAVA_HOME\bin;$env:PATH"
.\gradlew.bat :app:assembleDebug
```

工作目录：

```text
engines/android
```

## 当前运行范围

App/UI、EngineService、HTTP 通讯、脚本运行、停止、日志保留。

当前已完成脚本 API：

```lua
print(...)
m.sleep(ms)
m.systemTime()
m.tickCount()
m.touchDown(id, x, y)
m.touchMove(id, x, y)
m.touchUp(id)
m.keyDown(keycode)
m.keyUp(keycode)
m.keyPress(keycode)
m.inputText(text)
m.ime.lock()
m.ime.setText(text)
m.ime.unlock()
m.capture()
m.keepCapture()
m.releaseCapture()
m.setCaptureCacheMs(ms)
m.findColors(x1, y1, x2, y2, dir, sim, colors)
```
