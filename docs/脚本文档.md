# 小鱼精灵脚本文档

这是仓库和 GitHub 中稳定可阅读的脚本文档入口。每个 API 独立存放，便于按功能维护、审阅和
链接；需要目录搜索和参数摘要时，可打开 [交互式脚本文档](脚本文档.html)。

脚本默认启用小鱼精灵 `m` API，因此示例优先使用 `appIsFront()`、`capture()`、
`thread.newThread()` 这类实际调用写法。交互式文档右侧标题保留 `m.xxx`，用于标识实现归属；
显式写成 `m.xxx()` 也始终有效。

## 维护规则

1. 新增公开 API 时，同时新增或更新对应函数页、`catalog.json` 和 [API 总览](脚本文档/md/00-概述/API总览.md)。
2. 函数页开头必须保留 `params`、`returns` 元数据，参数需要写出名称和类型。
3. 提交前运行以下校验，确保目录、文档文件和元数据一致：

```powershell
.\tools\检查脚本文档.ps1
```
