**方法名称：** 从当前截图生成字形点阵。

**语法：** `m.font.getFontPixel(x1, y1, x2, y2, color)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 字形选择范围坐标。 |
| `y1` | `integer` | 是 | 字形选择范围坐标。 |
| `x2` | `integer` | 是 | 字形选择范围坐标。 |
| `y2` | `integer` | 是 | 字形选择范围坐标。 |
| `color` | `string` | 是 | 前景颜色规则，例如 `"1A73E8-202020"`。前半段为 RGB，后半段为 RGB 容差。 |

| 返回值 | 说明 |
|---|---|
| `string` | 成功时返回 `宽$高$十六进制点阵`。 |
| `nil, string` | 范围内没有前景点或颜色格式无效时返回错误信息。 |

**详细说明：**

```lua
local pixel = assert(m.font.getFontPixel(68, 1235, 112, 1265, "1A73E8-202020"))
local record = "脚$" .. pixel

assert(m.font.setDict(0, record))
assert(m.font.useDict(0))
```

返回内容会自动裁掉纯背景边缘，因此选择框可以略大于实际字符。选择框内应只包含一个目标
字形；多个字符、图标或装饰线都会一起进入点阵。
