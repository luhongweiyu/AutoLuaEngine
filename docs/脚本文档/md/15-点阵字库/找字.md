**方法名称：** 查找首个点阵文字。

**语法：** `m.font.findStr(x1, y1, x2, y2, text, color, sim)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 查找范围坐标。 |
| `y1` | `integer` | 是 | 查找范围坐标。 |
| `x2` | `integer` | 是 | 查找范围坐标。 |
| `y2` | `integer` | 是 | 查找范围坐标。 |
| `text` | `string` | 是 | 要查找的目标文字，必须能被当前字库拆成字形标签。 |
| `color` | `string` | 是 | 前景颜色规则。 |
| `sim` | `number` | 是 | 点阵相似度，范围 `(0, 1]`。 |

| 返回值 | 说明 |
|---|---|
| `integer, integer` | 找到时返回目标首字左上角坐标。 |
| `nil` | 未找到时只返回 `nil`。 |
| `nil, string` | 字库未包含目标字符或调用失败时返回错误信息。 |

**详细说明：**

```lua
local x, y, errorMessage = m.font.findStr(
    0, 1200, 720, 1279,
    "脚本",
    "1A73E8-202020",
    0.95
)

if x then
    print("找到脚本：", x, y)
elseif errorMessage then
    print("找字失败：", errorMessage)
end
```

找字只扫描 `text` 对应的字形，不会遍历整套字库。多字目标要求字符位于同一行且间隙符合
正常文字排版范围。
