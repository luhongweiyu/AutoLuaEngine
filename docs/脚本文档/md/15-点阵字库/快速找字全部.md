**方法名称：** 快速查找全部点阵文字。

**语法：** `m.font.findStrFastEx(x1, y1, x2, y2, text, color, sim)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 查找范围坐标。 |
| `y1` | `integer` | 是 | 查找范围坐标。 |
| `x2` | `integer` | 是 | 查找范围坐标。 |
| `y2` | `integer` | 是 | 查找范围坐标。 |
| `text` | `string` | 是 | 目标文字，必须能由当前字库中的一个或多个标签完整拆分。 |
| `color` | `string` | 是 | 前景颜色规则，格式 `RRGGBB-RRGGBB`。 |
| `sim` | `number` | 是 | 完整点阵相似度，范围 `(0, 1]`。 |

| 返回值 | 说明 |
|---|---|
| `string` | 返回 `x,y|x,y`；没有命中时返回空字符串。 |
| `nil, string` | 目标无法由字库标签拆分或调用失败时返回错误信息。 |

**详细说明：**

```lua
local points = assert(m.font.findStrFastEx(
    0, 0, 1079, 1919,
    "确定",
    "FFFFFF-202020",
    0.95
))
print(points)
```

匹配规则与 `m.font.findStrFast` 相同，坐标按从上到下、从左到右排序。该方法返回全部命中；
只需要第一处时使用 `m.font.findStrFast`，需要完整识字语义时使用 `m.font.findStrEx`。
