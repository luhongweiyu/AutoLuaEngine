**方法名称：** 选择当前线程使用的点阵字库。

**语法：** `m.font.useDict(index)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `index` | `integer` | 是 | 已通过 `m.font.setDict` 或 `m.font.addDict` 加载的字库索引。 |

| 返回值 | 说明 |
|---|---|
| `boolean` | 选择成功时返回 `true`。 |
| `nil, string` | 指定字库未加载或索引无效时返回错误信息。 |

**详细说明：**

```lua
assert(m.font.setDict(2, "/sdcard/xiaoyv/scripts/font.txt"))
assert(m.font.useDict(2))
```

字库选择是 Lua native 线程本地状态。主线程和每个 `m.thread` 子线程都需要各自调用
`useDict`，这样不同任务可以同时使用不同字库索引。
