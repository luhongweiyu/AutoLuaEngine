**方法名称：** 以结构化结果识别点阵文字。

**语法：** `m.font.read(x1, y1, x2, y2, color, sim)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 识字范围坐标。 |
| `y1` | `integer` | 是 | 识字范围坐标。 |
| `x2` | `integer` | 是 | 识字范围坐标。 |
| `y2` | `integer` | 是 | 识字范围坐标。 |
| `color` | `string` | 是 | 前景颜色规则，格式 `RRGGBB-RRGGBB`。 |
| `sim` | `number` | 是 | 点阵 Jaccard 相似度，范围 `(0, 1]`。 |

| 返回值 | 说明 |
|---|---|
| `table` | 成功时返回 `{ text = string, items = table }`；每项有 `text, x, y, w, h, score`。 |
| `nil, string` | 字库未选择、参数或截图失败时返回错误信息。 |

**详细说明：**

```lua
local result, errorMessage = m.font.read(0, 1200, 720, 1279, "1A73E8-202020", 0.95)
if not result then
    print("点阵识字失败：", errorMessage)
    return
end

for _, item in ipairs(result.items) do
    print(item.text, item.x, item.y, item.score)
end
```

该方法会扫描当前字库全部字形。候选位置由 11 位索引取得，最终分数由完整点阵中缺失点和多余
点共同计算。字库很大且只需要查找固定文本时，优先使用 `m.font.findStrFast`。
