params: "x1, y1, x2, y2: integer, color: string, sim: number"
returns: "string 或 nil, string"

**方法名称：** 识别点阵文字并返回坐标文本。

**语法：** `m.font.ocrEx(x1, y1, x2, y2, color, sim)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 识字范围坐标。 |
| `y1` | `integer` | 是 | 识字范围坐标。 |
| `x2` | `integer` | 是 | 识字范围坐标。 |
| `y2` | `integer` | 是 | 识字范围坐标。 |
| `color` | `string` | 是 | 前景颜色规则。 |
| `sim` | `number` | 是 | 点阵相似度，范围 `(0, 1]`。 |

| 返回值 | 说明 |
|---|---|
| `string` | 返回 `文字,x,y|文字,x,y`；没有命中时返回空字符串。 |
| `nil, string` | 失败时返回错误信息。 |

**详细说明：**

```lua
local items = assert(m.font.ocrEx(0, 1200, 720, 1279, "1A73E8-202020", 0.95))
print(items)
-- 例如：脚,75,1241|本,121,1241
```

该接口使用与 `m.font.ocr` 相同的直接点阵匹配结果，只是按 `|` 分隔输出每个字的文字和左上角
坐标。需要避免字符串解析时可改用 `m.font.read`。
