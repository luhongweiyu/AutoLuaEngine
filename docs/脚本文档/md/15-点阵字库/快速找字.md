**方法名称：** 快速查找首个点阵文字。

**语法：** `m.font.findStrFast(x1, y1, x2, y2, text, color, sim)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 查找范围坐标。 |
| `y1` | `integer` | 是 | 查找范围坐标。 |
| `x2` | `integer` | 是 | 查找范围坐标。 |
| `y2` | `integer` | 是 | 查找范围坐标。 |
| `text` | `string` | 是 | 目标文字，必须能由当前字库中的一个或多个标签完整拆分。 |
| `color` | `string` | 是 | 前景颜色规则，格式 `RRGGBB-RRGGBB`。 |
| `sim` | `number` | 是 | 完整点阵相似度，范围 `(0, 1]`。 |

| 返回值 | 说明 |
|---|---|
| `integer, integer` | 找到时返回目标首字左上角坐标。 |
| `nil` | 未找到时只返回 `nil`。 |
| `nil, string` | 目标无法由字库标签拆分或调用失败时返回错误信息。 |

**详细说明：**

```lua
local x, y = m.font.findStrFast(
    0, 1200, 720, 1279,
    "立即运行",
    "FFFFFF-202020",
    0.95
)
```

该方法只搜索 `text` 涉及的字形模板，再按同行和正常字间距连接结果，不扫描字库中的无关字形。
它适合大字库中的固定按钮、状态文字和短标签；若必须确认目标字符在完整识字结果中连续出现，
使用 `m.font.findStr`。
