**方法名称：** 识别点阵文字。

**语法：** `m.font.ocr(x1, y1, x2, y2, color, sim)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 识字范围坐标。 |
| `y1` | `integer` | 是 | 识字范围坐标。 |
| `x2` | `integer` | 是 | 识字范围坐标。 |
| `y2` | `integer` | 是 | 识字范围坐标。 |
| `color` | `string` | 是 | 前景颜色规则。 |
| `sim` | `number` | 是 | 点阵相似度，范围 `(0, 1]`。 |

| 返回值 | 说明 |
|---|---|
| `string` | 识别成功时返回文字；没有命中时返回空字符串。 |
| `nil, string` | 字库、参数或截图失败时返回错误信息。 |

**详细说明：**

```lua
local text, errorMessage = m.font.ocr(0, 1200, 720, 1279, "1A73E8-202020", 0.95)
if not text then
    print("识字失败：", errorMessage)
else
    print("识别结果：", text)
end
```

这是与大漠类点阵识字习惯一致的字符串接口。需要每个字的坐标和分数时使用 `m.font.read`。
