**方法名称：** 设置点阵字库。

**语法：** `m.font.setDict(index, dictionary)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `index` | `integer` | 是 | 字库索引，范围 `0` 到 `255`。 |
| `dictionary` | `string` | 是 | 多行字库文本、普通文件路径、当前 `.alpkg` 资源路径，或一条单独字库记录。 |

| 返回值 | 说明 |
|---|---|
| `boolean` | 设置成功时返回 `true`。 |
| `nil, string` | 格式、路径或字形数据无效时返回错误信息。 |

**详细说明：**

```lua
local dict = [[
# 常用蓝色底部导航字形
脚$31$16$01000100F238...
本$30$16$...
]]

assert(m.font.setDict(0, dict))
assert(m.font.useDict(0))
```

该方法会完全替换指定索引原有字库。字库数据在 native 内按不可变快照保存，正在运行的其他
Lua native 子线程不会读到被重分配的旧内存。加载时会为全部字形建立 11 位候选索引和标签
索引；最终识别仍比较每个候选的完整宽高点阵。
