**方法名称：** 查找全部点阵文字。

**语法：** `m.font.findStrEx(x1, y1, x2, y2, text, color, sim)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `x1` | `integer` | 是 | 查找范围坐标。 |
| `y1` | `integer` | 是 | 查找范围坐标。 |
| `x2` | `integer` | 是 | 查找范围坐标。 |
| `y2` | `integer` | 是 | 查找范围坐标。 |
| `text` | `string` | 是 | 目标文字。 |
| `color` | `string` | 是 | 前景颜色规则。 |
| `sim` | `number` | 是 | 点阵相似度，范围 `(0, 1]`。 |

| 返回值 | 说明 |
|---|---|
| `string` | 返回 `x,y|x,y`；没有命中时返回空字符串。 |
| `nil, string` | 字库、参数或截图失败时返回错误信息。 |

**详细说明：**

```lua
local points = assert(m.font.findStrEx(
    0, 0, 1079, 1919,
    "确定",
    "FFFFFF-202020",
    0.95
))
print(points)
-- 例如：120,800|620,800
```

该方法先完整识别区域内当前字库的全部字形，再返回每处连续目标文本的首字坐标。坐标按从上
到下、从左到右排序。大字库固定文本搜索需要更低耗时时，使用 `m.font.findStrFastEx`。
