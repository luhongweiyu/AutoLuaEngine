**方法名称：** 向点阵字库追加字形。

**语法：** `m.font.addDict(index, dictionary)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `index` | `integer` | 是 | 字库索引，范围 `0` 到 `255`。 |
| `dictionary` | `string` | 是 | 追加的字库文本或字库文件路径。 |

| 返回值 | 说明 |
|---|---|
| `boolean` | 追加成功时返回 `true`。 |
| `nil, string` | 字库索引、文件或字形格式无效时返回错误信息。 |

**详细说明：**

```lua
assert(m.font.setDict(0, "脚$31$16$01000100F238..."))
assert(m.font.addDict(0, "本$30$16$..."))
assert(m.font.useDict(0))
```

追加会生成新的字库快照再替换旧快照，适合按需补充少量字符。重复标签不会自动覆盖，识别时按
匹配分数选择结果；需要重建字库时使用 `setDict`。
