# 选择控件与表格

索引统一从 `0` 开始。组合框项目既可使用竖线字符串，也可直接传 Lua 数组；字符串中的
竖线可写成 `\|`。

## 组合框与单选组

| 方法 | 返回值 | 说明 |
|---|---|---|
| `createComboBox(parent, items[, width])` | `integer` | `items` 为 `"甲|乙"` 或 `{ "甲", "乙" }` |
| `createRadioGroup(parent, label)` | `integer?` | 创建单选按钮组 |
| `addOptionItem(handle, text)` | 无 | 向组合框追加选项，不改变当前选择 |
| `addRadioBox(handle, text[, wrapLine])` | `boolean` | 追加单选项；`wrapLine=true` 让下一项另起一行 |
| `getItemText(handle, index)` | `string?` | 返回指定选项文本 |
| `removeItemAt(handle, index)` | 无 | 删除指定选项 |
| `removeAllItems(handle)` | 无 | 清空全部选项并取消选择 |
| `getSelectedItemIndex(handle)` | `integer` | 未选择返回 `-1`；无效句柄返回 `-2` |
| `setItemSelected(handle, index)` | 无 | 修改选择并触发选择回调 |
| `getItemCount(handle)` | `integer` | 返回选项数；错误返回 `-1` |
| `setOnSelectEvent(handle, callback)` | 无 | `function(handle, index, text)`，传 `nil` 移除 |

## 表格

| 方法 | 返回值 | 说明 |
|---|---|---|
| `createTableView(parent, title, columns, showHeader[, w[, h]])` | `integer?` | 创建可滚动、可调整列宽的表格 |
| `setTableHeaderItem(handle, column, text)` | 无 | 设置表头文本 |
| `insertTableRow(handle, after)` | `integer?` | 返回新行索引；`-1` 插到开头，`-2` 追加末尾 |
| `getTableItemText(handle, row, column)` | `string?` | 读取单元格 |
| `setTableItemText(handle, row, column, text)` | 无 | 修改单元格 |
| `deleteTableRow(handle, row)` | 无 | 删除指定行并修正选择索引 |
| `clearTable(handle)` | 无 | 清空行，保留列数和表头 |
| `getItemCount(handle)` | `integer` | 对表格返回行数 |
| `getSelectedItemIndex(handle)` | `integer` | 对表格返回已选行 |
| `setItemSelected(handle, row)` | 无 | 选择指定行的第 `0` 列，并触发表格选择回调 |
| `setOnSelectEventEx(handle, callback)` | 无 | `function(handle, row, column, text)` |

```lua
local tableView = imgui.createTableView(parent, "状态", 2, true, -1, 240)
imgui.setTableHeaderItem(tableView, 0, "名称")
imgui.setTableHeaderItem(tableView, 1, "结果")
local row = imgui.insertTableRow(tableView, -2)
imgui.setTableItemText(tableView, row, 0, "网络")
imgui.setTableItemText(tableView, row, 1, "正常")
```
