# 图片、样式与图形

## 图片与控件状态

| 方法 | 返回值 | 说明 |
|---|---|---|
| `createImage(parent[, path[, w[, h]]])` | `integer?` | 路径为空时创建空图片控件 |
| `setImage(handle, path)` | 无 | 普通文件、脚本相对路径和 ALPKG 资源均可 |
| `setImageFromBitmap(handle, bitmap)` | 无 | 接受 `android.graphics.Bitmap` |
| `setWidgetSize(handle, w, h)` | 无 | `0` 自动、`-1` 填充可用空间 |
| `setWidgetVisible(handle, visible)` | 无 | 设置任意控件可见性 |
| `isWidgetVisible(handle)` | `boolean` | 无效句柄也返回 `false`，错误另取 |
| `setWidgetStyle(handle, style, v1[, v2])` | 无 | 设置该控件的 Dear ImGui StyleVar |
| `setWidgetColor(handle, type, color)` | 无 | 设置该控件的 Dear ImGui 颜色 |

路径图片在 SO 内解码成紧凑 RGBA8888；Java `Bitmap` 在 Lua-Java 边界复制成同样格式。
渲染快照使用共享所有权，脚本替换图片时当前帧不会读取已释放内存。GL 纹理由渲染线程
创建，控件删除、图片替换或脚本结束后自动释放。

样式常量使用 `imgui.StyleVar`，颜色常量使用 `imgui.Color`：

```lua
imgui.setWidgetStyle(button, imgui.StyleVar.FrameRounding, 8)
imgui.setWidgetColor(button, imgui.Color.Button, 0xFF167D72)
```

兼容别名为 `imgui.ImGuiStyleVar`、`imgui.ImGuiColor` 和 `imgui.ImGuiCol`。常量值直接
取自当前编入 SO 的 Dear ImGui `v1.92.8`，无需脚本手写数字。

## 图形

图形直接绘制在当前 Surface 的屏幕坐标上，不属于窗口布局，也不会截获触摸。

| 方法 | 返回值 | 说明 |
|---|---|---|
| `createRectangle(x, y, x2, y2, color, filled, rounding)` | `integer?` | 创建矩形 |
| `createCircle(x, y, radius, color, filled, segments)` | `integer?` | `segments` 至少为 `3` |
| `createPolygon(points, color, closed, filled, thickness)` | `integer?` | 支持嵌套点表或平面坐标数组 |
| `createLine(x1, y1, x2, y2, color, thickness)` | `integer?` | 创建线段 |
| `createBitmapShape(x, y, w, h, bitmapOrPath)` | `integer?` | 接受路径或 Java `Bitmap` |
| `createShapeText(x, y, w, h, text, textColor, bgColor, hasBg, scale)` | `integer` | 创建带裁剪区域的文本 |
| `setShapePosition(handle, x, y)` | `integer` | 成功 `0`、失败 `-1`；整体平移图形 |
| `setShapeVisibility(handle, visible)` | `integer` | 成功 `0`、失败 `-1` |
| `isShapeVisibility(handle)` | `boolean` | 查询图形是否可见 |
| `setShapeTextString(handle, text)` | `boolean` | 修改文本图形内容 |
| `setShapeTextColor(handle, color)` | `boolean` | 修改文字颜色 |
| `setShapeTextBackground(handle, color, hasBg)` | `boolean` | 修改背景及显示状态 |
| `setShapeTextFontScale(handle, scale)` | `boolean` | 缩放必须大于 `0` |
| `setBitmapShape(handle, bitmapOrPath)` | 无 | 替换位图图形内容 |
| `setShapeThickness(handle, thickness)` | `boolean` | 修改边框或线段粗细 |
| `removeShape(handle)` | `integer` | 成功 `0`、失败 `-1` |

`points` 支持以下三种等价格式：

```lua
{{10, 10}, {100, 20}, {60, 90}}
{{x = 10, y = 10}, {x = 100, y = 20}, {x = 60, y = 90}}
{10, 10, 100, 20, 60, 90}
```
