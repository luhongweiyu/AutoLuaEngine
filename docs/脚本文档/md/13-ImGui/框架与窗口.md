# 框架与窗口

## 方法总表

| 方法 | 参数 | 返回值 | 说明 |
|---|---|---|---|
| `imgui.isSupport()` | 无 | `boolean` | 检查 OpenGL ES 3 和 Android 宿主能力 |
| `imgui.show([touchable], [font], [fontSize])` | 类型可交换 | `boolean` | 显示全屏透明 Surface；可触摸时阻塞至关闭 |
| `imgui.showWindow(config)` | `config: table` | `boolean` | 显示独立悬浮 Surface，立即返回 |
| `imgui.close()` | 无 | 无 | 关闭当前 ImGui 框架，可重复调用 |
| `imgui.setColorTheme(style)` | `style: integer` | `boolean` | `1` 浅色、`2` 经典、其他值深色 |
| `imgui.createWindow(title, x, y, w, h, showClose)` | 文本、坐标、尺寸、布尔值 | `integer` | 创建普通 Dear ImGui 窗口 |
| `imgui.destroyWindow(handle)` | `handle: integer` | 无 | 销毁窗口及其全部子控件 |
| `imgui.setOnClose(handle, callback)` | 句柄、函数或 `nil` | 无 | 设置或移除窗口关闭回调 |
| `imgui.setWindowPos(handle, x, y)` | 句柄、坐标 | 无 | 设置普通窗口左上角 |
| `imgui.setWindowSize(handle, w, h)` | 句柄、正数尺寸 | 无 | 设置普通窗口尺寸 |
| `imgui.getWindowPos(handle)` | `handle: integer` | `table?` | 返回 `{x,y,width,height}`，失败返回 `nil` |
| `imgui.setWindowFlags(handle, flags)` | 句柄、整数位标志 | 无 | 设置 Dear ImGui 窗口标志 |
| `imgui.getLastError()` | 无 | `string` | 当前线程最近一次 ImGui 错误 |
| `imgui.isValidHandle(handle)` | `handle: integer` | `boolean` | 检查窗口、控件或图形句柄 |

## `show` 参数

三个可选参数按类型识别，顺序不限，但同一类型不能重复：

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `touchable` | `boolean` | `true` | `true` 接收触摸并阻塞；`false` 穿透触摸并立即返回 |
| `font` | `string` | 系统字体 | 字体文件路径；加载失败自动使用 Android 系统字体 |
| `fontSize` | `number` | `45` | 内容字体物理像素大小 |

```lua
imgui.show(true, "/sdcard/fonts/custom.ttf", 32)
imgui.show(32, false) -- 顺序可交换，画面不可触摸并立即返回
```

## `showWindow` 配置

| 字段 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `x` / `y` | `integer` | `0` | 外层悬浮窗口左上角 |
| `width` / `height` | `integer` | `600` | 外层悬浮窗口尺寸，必须大于 `0` |
| `hastitle` | `boolean` | `true` | 是否显示可拖动标题栏 |
| `title` | `string` | `""` | 标题文字 |
| `titlecolor` | `integer` | `0xFFFFFFFF` | 标题文字颜色 |
| `titlebgcolor` | `integer` | `0xFF87CEFA` | 标题栏背景颜色 |
| `hasclose` | `boolean` | `true` | 是否显示关闭按钮 |
| `closecolor` | `integer` | `0xFFFFFFFF` | 关闭图标颜色 |
| `hasresize` | `boolean` | `true` | 是否显示右下角缩放柄 |
| `resizecolor` | `integer` | `0xFFFFFFFF` | 缩放柄颜色 |
| `hastoggle` | `boolean` | `true` | 是否显示收起/展开按钮 |
| `togglecolor` | `integer` | `0xFFFFFFFF` | 收起/展开图标颜色 |
| `fontsize` | `number` | `100` | 标题字体大小 |
| `font` | `string` | 系统字体 | 内容字体路径，小鱼精灵扩展字段 |
| `contentfontsize` | `number` | `45` | 内容字体大小，小鱼精灵扩展字段 |

```lua
local ok = imgui.showWindow({
    x = 80,
    y = 160,
    width = 560,
    height = 760,
    title = "运行面板",
    fontsize = 34,
    contentfontsize = 28,
})
```

`showWindow` 非阻塞；主脚本必须继续运行，否则脚本结束清理会立即关闭窗口。标题栏收起时
外层 WindowManager 高度也会缩到标题栏，不会用透明区域拦截下方应用触摸。

调用 `showWindow` 时，第一个已创建的 ImGui 内容窗口默认调整为
`{x=0, y=0, width=config.width, height=config.height}`，填满外层画布；显示成功后仍可通过
`setWindowPos` 和 `setWindowSize` 再次修改。

## 窗口标志

使用 `imgui.WindowFlags`，多个标志用按位或组合：

```lua
local flags = imgui.WindowFlags.NoResize | imgui.WindowFlags.NoCollapse
imgui.setWindowFlags(window, flags)
```

同时提供兼容别名 `imgui.ImGuiWindowFlags`。公开字段包括 `NoTitleBar`、`NoResize`、
`NoMove`、`NoScrollbar`、`NoScrollWithMouse`、`NoCollapse`、`AlwaysAutoResize`、
`NoBackground`、`NoMouseInputs`、`HorizontalScrollbar`、`NoNavInputs`、`NoNavFocus`、
`NoDecoration` 和 `NoInputs` 等 Dear ImGui 公共窗口标志。
