# ImGui 概述

小鱼精灵内置官方 Dear ImGui `v1.92.8`。Lua 使用全局 `imgui` 表；控件模型、事件队列、
图片和图形能力均位于 `libengine.so`，后续 JS、Go 与插件复用同一组 `engine_imgui*` C ABI。

当前已经实现懒人精灵 ImGui 文档列出的全部 `82` 个方法，并额外提供
`getLastError`、`isValidHandle`、`post` 三个运行辅助方法。兼容规则如下：

- 句柄是大于 `0` 的 `integer`，创建失败时按原方法契约返回 `0` 或 `nil`。
- 组合框、单选组及表格的项、行、列索引均从 `0` 开始。
- 颜色格式固定为 `0xAARRGGBB`；也接受等价的有符号 32 位整数，例如 `-1`。
- 坐标与尺寸单位均为屏幕物理像素。
- 所有错误可在调用后通过 `imgui.getLastError()` 读取。
- `imgui.show(true, ...)` 阻塞当前任务；回调由内部事件泵取得 VM Gate 后执行，不在
  Android UI 或 OpenGL 渲染线程中执行。

## 最小示例

```lua
local window = imgui.createWindow("示例", 40, 100, 620, 500, true)
local layout = imgui.createVerticalLayout(window, -1, -1)
local input = imgui.createInputText(layout, "名称", "小鱼精灵", 0, -1, 0)
local close = imgui.createButton(layout, "关闭", 180, 56)

imgui.setOnClick(close, function()
    print(imgui.getInputText(input))
    imgui.close()
end)

local ok = imgui.show(true, 30)
if not ok then
    error(imgui.getLastError())
end
```

完整可运行示例位于 App 内置脚本 `imgui.lua`，覆盖中文、输入框、选择、表格、Java
`Bitmap`、图形和回调。

## 调用链

```text
Lua imgui.*
  -> runtime/lua/imgui_lua_api
  -> engine_imgui* C ABI
  -> core/api/imgui_api
  -> platform/imgui_renderer
  -> :engine/ScriptImGuiService
```

未调用 `show` 或 `showWindow` 时，不创建 Service、Surface、EGL context 或渲染线程。
