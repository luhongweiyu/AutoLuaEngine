# 布局与基础控件

所有子控件都需要有效父句柄。普通控件可放入窗口、垂直布局、水平布局、树形布局或标签页。
尺寸 `0` 表示自动计算，`-1` 表示使用父容器当前可用空间。

## 布局

| 方法 | 返回值 | 说明 |
|---|---|---|
| `createVerticalLayout(parent[, w[, h]])` | `integer`，失败 `0` | 子项按添加顺序纵向排列 |
| `createHorticalLayout(parent[, w[, h]])` | `integer`，失败 `0` | 子项横向排列；保留原 API 的拼写 |
| `createTreeBoxLayout(parent, title, w)` | `integer?` | 创建默认展开的树形容器 |
| `createTabBar(parent, title)` | `integer?` | 创建标签栏 |
| `addTabBarItem(tabBar, title)` | `integer?` | 添加标签页并返回可作为父容器的句柄 |
| `sameLine(handle, spacing)` | `boolean` | 让控件与前一个控件同行，`-1` 使用系统间距 |
| `setLayoutBorderVisible(handle, visible)` | 无 | 设置垂直/水平布局边框 |

## 控件

| 方法 | 返回值 | 说明 |
|---|---|---|
| `createButton(parent, text, w, h)` | `integer` | 推荐的父容器按钮形式 |
| `createButton(x, y, w, h, text)` | `integer` | 全屏画布绝对坐标兼容形式 |
| `setOnClick(handle, callback)` | 无 | 回调参数为 `handle`；传 `nil` 移除 |
| `createLabel(parent, text[, singleLine])` | `integer` | `singleLine` 默认 `true` |
| `createCheckBox(parent, label[, checked])` | `integer` | 创建复选框 |
| `createSwitch(parent, label[, checked[, h]])` | `integer` | 创建移动端开关 |
| `setChecked(handle, checked)` | 无 | 修改复选框/开关并触发回调 |
| `isChecked(handle)` | `boolean?` | 返回状态；无效句柄返回 `nil` |
| `setOnCheck(handle, callback)` | 无 | 回调为 `function(handle, checked)` |
| `createInputText(parent, label[, value[, type[, w[, h]]]])` | `integer` | 创建普通、密码或多行输入框 |
| `getInputText(handle)` | `string?` | 读取真实输入文本 |
| `setInputText(handle, text)` | 无 | 替换输入文本 |
| `setInputType(handle, type)` | 无 | `0` 普通、`1` 密码、`2` 多行；类型变化时清空文本 |
| `createProgressBar(parent, progress[, w[, h]])` | `integer` | `progress` 范围 `0.0` 到 `1.0` |
| `setProgressBarPos(handle, progress)` | 无 | 超出范围时自动限制到 `0.0..1.0` |
| `getProgressBarPos(handle)` | `number?` | 返回当前进度 |
| `createSlider(parent, label, min, max, initial[, w])` | `integer?` | 创建整数滑动条 |
| `setSlider(handle, position)` | 无 | 修改位置并触发滑动回调 |
| `getSliderPos(handle)` | `integer?` | 返回当前整数位置 |
| `setOnSliderEvent(handle, callback)` | 无 | 回调为 `function(handle, position)` |
| `createColorPicker(parent[, title[, color[, w[, h]]]])` | `integer?` | 创建带 Alpha 的颜色选择器 |

输入类型也可使用 `imgui.InputType.Text`、`Password`、`Multiline`，避免在脚本中散落数字。

```lua
local row = imgui.createHorticalLayout(parent, -1, 64)
local save = imgui.createButton(row, "保存", 180, 54)
local cancel = imgui.createButton(row, "取消", 180, 54)
imgui.sameLine(cancel, 12)
```
