# ImGui 常量表

常量数值直接取自项目编入的 Dear ImGui `v1.92.8`，脚本应使用名称，不要自行抄写会随版本
变化的 StyleVar 和 Color 数字。

## 输入类型和主题

| 常量 | 说明 |
|---|---|
| `imgui.InputType.Text` | 普通单行文本，值为 `0`。 |
| `imgui.InputType.Password` | 密码输入，值为 `1`。 |
| `imgui.InputType.Multiline` | 多行文本，值为 `2`。 |
| `imgui.Theme.Dark` | 暗色主题，值为 `0`。 |
| `imgui.Theme.Light` | 浅色主题，值为 `1`。 |
| `imgui.Theme.Classic` | 经典主题，值为 `2`。 |

## 窗口标志

`imgui.WindowFlags` 提供 `None`、`NoTitleBar`、`NoResize`、`NoMove`、`NoScrollbar`、
`NoScrollWithMouse`、`NoCollapse`、`AlwaysAutoResize`、`NoBackground`、`NoSavedSettings`、
`NoMouseInputs`、`MenuBar`、`HorizontalScrollbar`、`NoFocusOnAppearing`、
`NoBringToFrontOnFocus`、`AlwaysVerticalScrollbar`、`AlwaysHorizontalScrollbar`、`NoNavInputs`、
`NoNavFocus`、`UnsavedDocument`、`NoNav`、`NoDecoration` 和 `NoInputs`。

多个标志使用 Lua 5.4 按位或运算符组合：

```lua
local flags = imgui.WindowFlags.NoResize | imgui.WindowFlags.NoCollapse
imgui.setWindowFlags(window, flags)
```

兼容别名为 `imgui.ImGuiWindowFlags`。

## 样式和颜色

`imgui.StyleVar` 包含当前 Dear ImGui 的全部公共 `ImGuiStyleVar` 名称，例如
`WindowPadding`、`WindowRounding`、`FramePadding`、`FrameRounding`、`ItemSpacing`、
`CellPadding`、`GrabRounding` 和 `ButtonTextAlign`。二维样式必须同时传入 `v1`、`v2`：

```lua
imgui.setWidgetStyle(button, imgui.StyleVar.FramePadding, 16, 8)
imgui.setWidgetStyle(button, imgui.StyleVar.FrameRounding, 8)
```

`imgui.Color` 包含当前 Dear ImGui 的全部公共 `ImGuiCol` 名称，例如 `Text`、`WindowBg`、
`FrameBg`、`Button`、`ButtonHovered`、`ButtonActive`、`Header`、`CheckMark` 和 `TableRowBg`：

```lua
imgui.setWidgetColor(button, imgui.Color.Button, 0xFF167D72)
```

兼容别名为 `imgui.ImGuiStyleVar`、`imgui.ImGuiColor` 和 `imgui.ImGuiCol`。颜色统一使用
`0xAARRGGBB`，其中 `AA` 是透明度。
