**方法名称：** 发送短信。

**语法：** `sendSms(number, content)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `number` | `string` | 是 | 具体取值和组合规则见下方详细说明。 |
| `content` | `string` | 是 | 具体取值和组合规则见下方详细说明。 |

| 返回值 | 说明 |
|---|---|
| 无 | 此方法不返回值。 |

**使用示例：**

```lua
sendSms("10086", "示例")
```

**详细说明：**

向 `number` 发送文本 `content`，需要真实电话能力。RootDaemon 初始化时会一次性授予应用
所需运行时权限。
