`m.dialog.*` 和 `m.ui.form` 使用 WindowManager 悬浮窗口，和旧项目脚本 UI 一样不会启动 Activity：不会压暗屏幕，不启动或带回
小鱼精灵 页面。点击框外区域会直接穿透到下方应用且不会关闭对话框；只能点击界面自身的
确认/取消按钮，或由脚本停止、引擎关闭时自动关闭。

原生弹窗、表单和 HUD 都需要 App 已取得悬浮窗权限；权限未开启时创建函数返回
`nil, errorMessage:string`。

## 方法一览

| 方法 | 参数类型 | 成功/用户操作返回值 | 失败返回值 |
|---|---|---|---|
| `m.dialog.alert` | `title:string, message:string, buttonText:string?` | `boolean` | `nil, errorMessage:string` |
| `m.dialog.confirm` | `title:string, message:string, positiveText:string?, negativeText:string?` | `boolean` | `nil, errorMessage:string` |
| `m.dialog.input` | `title:string, hint:string, defaultText:string?, options:table?` | `text:string`；取消为 `nil` | `nil, errorMessage:string` |
| `m.dialog.select` | `title:string, items:table<string>, selectedIndex:integer?, options:table?` | `index:integer, value:string`；取消为 `nil` | `nil, errorMessage:string` |
| `m.ui.form` | `spec:table` | `values:table`；取消为 `nil` | `nil, errorMessage:string` |

`m.dialog.select` 的索引从 `1` 开始。`options` 中的 `message`、`inputType`、
`positiveText`、`negativeText` 是 `string`，`multiline`、`selectAll` 是 `boolean`。

## input / select 的 options

| 字段 | 类型 | 说明 |
|---|---|---|
| `message` | `string` | 弹窗说明文字 |
| `multiline` | `boolean` | 是否允许多行输入，仅输入框有效 |
| `inputType` | `string` | `text`、`number` 或 `password` |
| `selectAll` | `boolean` | 获得焦点时是否全选，仅输入框有效 |
| `positiveText` | `string` | 确认按钮文字 |
| `negativeText` | `string` | 取消按钮文字 |

表单字段 `type` 支持：`text`、`password`、`number`、`multiline`、`boolean`、`select`、`label`。
每个字段的 `name`、`label`、`type`、`hint` 是 `string`，`items` 是 `table<string>`，`selectedIndex` 是 `integer`；`default` 按字段类型使用 `string` 或 `boolean`。
确认后 `values` 以 `name` 为键（如 `values.account`、`values.enabled`、`values.mode` 为选中文字）。
具体命令见左侧各项。


## 返回值要点（与主文档一致）

- `m.dialog.alert`：三个参数均为 `string`，`buttonText` 可省略；确认或关闭返回 `boolean`；创建失败返回 `nil, errorMessage:string`。
- `m.dialog.confirm`：后两个按钮文案可省略；确认 `true`，取消或关闭 `false`；创建失败 `nil, errorMessage:string`。
- `m.dialog.input`：确认返回 `text:string`，取消 `nil`，创建失败 `nil, errorMessage:string`。
- `m.dialog.select`：确认返回 `index:integer, value:string`（索引从 1 起），取消 `nil`。
- `m.ui.form`：确认返回 `values:table`，取消 `nil`，创建失败 `nil, errorMessage:string`。
