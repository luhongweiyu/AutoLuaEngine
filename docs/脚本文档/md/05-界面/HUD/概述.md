## 示例

```lua
local handle, errorMessage = m.hud.show("status", {
    text = "脚本正在运行",
    x = 24,
    y = 180,
    w = 260,
    h = 72,
    draggable = true,
    buttons = {
        { id = "stop", text = "停止" },
    },
})
assert(handle, errorMessage)

local event = m.hud.waitEvent("status", 30000)
if event and event.type == "click" and event.data.id == "stop" then
    print("用户点击停止")
end
m.hud.hide("status")
```

## API

| 方法 | 参数类型 | 返回值类型 |
|---|---|---|
| `m.hud.show(id, spec)` | `id:any, spec:table` | `handle:integer` 或 `nil, errorMessage:string` |
| `m.hud.update(id, patch)` | `id:any, patch:table` | `boolean` 或 `nil, errorMessage:string` |
| `m.hud.hide(id)` | `id:any` | `boolean` 或 `nil, errorMessage:string` |
| `m.hud.waitEvent(id[, timeoutMs])` | `id:any, timeoutMs:integer?` | `event:table` 或 `nil, errorMessage:string` |
| `m.toast(text[, durationMs])` | `text:any, durationMs:integer?` | `handle:integer` 或 `nil, errorMessage:string` |

## 配置字段

| 字段 | 类型 | 说明 |
|---|---|---|
| `text` | `string` | HUD 主文字 |
| `x`、`y` | `integer` | 屏幕物理像素坐标 |
| `w`、`h` | `integer` | 固定尺寸；未传时按内容自适应 |
| `gravity` | `string` | `top_left`、`center` 或 `bottom` |
| `backgroundColor`、`textColor` | `string` | 颜色，`#RRGGBB` 或 `#AARRGGBB` |
| `textSize`、`alpha` | `number` | 字号与透明度 |
| `padding`、`radius` | `integer` | 内边距、圆角 |
| `durationMs` | `integer` | 自动关闭毫秒；`0` 或未传表示不自动关闭 |
| `clickable`、`draggable` | `boolean` | 可点、可拖 |
| `clickId` | `string` | 点击 HUD 本体时的 id，默认 `hud` |
| `buttons` | `table<table>` | 按钮数组，每项包含 `id:any` 和 `text:string` |

`durationMs` 大于 `0` 时 HUD 自动关闭并发送
`{ type = "closed", data = { reason = "timeout" } }`。

## 事件

```lua
{
    type = "click" | "closed" | "error" | "timeout",
    data = { ... },
}
```

- 点击 HUD 本体：`data.id` 为 `clickId`（默认 `hud`）
- 点击按钮：`data.id` / `data.text` 来自按钮配置
- `waitEvent` 的 `timeoutMs` 到期时，`type` 是 `timeout`，不关闭 HUD

需要悬浮窗权限；失败返回 `nil, errorMessage:string`。

`m.toast` 显示自动关闭提示，不阻塞脚本。

类型综述：`text`、`gravity`、颜色、`clickId` 为 `string`；`x`/`y`/`w`/`h`/`padding`/`radius`/`durationMs` 为 `integer`；`textSize`、`alpha` 为 `number`；`clickable`、`draggable` 为 `boolean`；`buttons` 为 `table<table>`。
