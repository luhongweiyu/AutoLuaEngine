Lua 仍是动态语言，下面的类型是 API 契约：`integer` 表示整数，`number` 表示整数或小数，
`string` 可承载文本或二进制数据，`table` 表示 Lua 表，`userdata` 表示 native/Java 对象，
`any` 表示任意 Lua 值，类型后的 `?` 表示该值可能为 `nil`。

列表形式：

- `integer` 表示整数
- `number` 表示整数或小数
- `string` 可承载文本或二进制数据
- `table` 表示 Lua 表
- `userdata` 表示 native/Java 对象
- `any` 表示任意 Lua 值
- 类型后的 `?` 表示该值可能为 `nil`
