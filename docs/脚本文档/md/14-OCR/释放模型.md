**方法名称：** 释放 OCR 模型名称。

**语法：** `m.ocr.release(name)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `name` | `string` | 是 | `m.ocr.loadBuiltin` 或 `m.ocr.load` 时使用的模型名称。 |

| 返回值 | 说明 |
|---|---|
| `true` | 已释放该名称持有的模型引用。 |
| `false` | 指定名称原本没有加载。 |
| `nil, string` | 调用失败时返回错误信息。 |

**详细说明：**

```lua
local released, errorMessage = m.ocr.release("main")
if released == nil then
    print("释放失败：", errorMessage)
elseif released then
    print("已释放 main")
end
```

同一套模型可以被多个名称共享。只有最后一个名称释放后，底层 ONNX session 才会关闭。
