**方法名称：** 查询 OCR 模型是否已加载。

**语法：** `m.ocr.isLoaded(name)`

**参数说明：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `name` | `string` | 是 | 模型名称。 |

| 返回值 | 说明 |
|---|---|
| `boolean` | 已加载返回 `true`，未加载返回 `false`。 |
| `nil, string` | 参数或平台调用异常时返回错误信息。 |

**详细说明：**

```lua
if not m.ocr.isLoaded("main") then
    print("OCR 模型尚未加载")
end
```

该方法只查询名称绑定状态，不会触发模型加载。
